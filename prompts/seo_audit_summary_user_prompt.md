# SEO Audit Summary User Prompt

You are conducting a comprehensive SEO audit. Follow these steps:

## 1. Initial Setup
- Confirm the website URL to audit
- Set crawl parameters (depth, limits, scope)
- Configure Screaming Frog settings for the audit

## 2. Crawl Execution
- Run Screaming Frog crawl using the CLI tool
- Monitor crawl progress and completion
- Export crawl data in appropriate formats

## 3. Analysis Areas

### Technical SEO
- Identify broken links (404s, 500s)
- Analyze redirect chains
- Check canonical tag implementation
- Review robots.txt and XML sitemaps
- Evaluate site speed and Core Web Vitals
- Check mobile-friendliness

### On-Page SEO
- Audit title tags (length, uniqueness, keywords)
- Review meta descriptions
- Analyze heading structure (H1-H6)
- Check image alt text
- Identify thin or duplicate content
- Review internal linking structure

### Indexability
- Find noindex/nofollow issues
- Identify orphaned pages
- Check URL structure and parameters
- Review hreflang implementation (if international)

## 4. Prioritization
Categorize issues by severity:
- **Critical**: Issues blocking indexing or causing major UX problems
- **High**: Significant SEO impact
- **Medium**: Optimization opportunities
- **Low**: Minor improvements

## 5. Report Generation
- Create executive summary
- Detail findings by category
- Provide actionable recommendations
- Export data to Google Sheets
- Generate visual reports with screenshots where relevant
- Store final report in Google Drive

## 6. Deliverables
- Comprehensive audit report (PDF/Google Doc)
- Google Sheet with all findings
- Screenshots of critical issues
- Prioritized action plan
- Share links for client access

Focus on providing clear, actionable insights that non-technical stakeholders can understand while maintaining technical accuracy.
